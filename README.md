## Locate Main.cpp and open Terminal Directory
```bash
C:\user\OS_EMULATOR
```

## Compile and Run (Linux)
Enter the following commands: 
```bash
g++ o- Main -I headers source/*.cpp
./Main
```

## Compile and Run (Mac)
Enter the following commands:
```bash
g++ -std=c++20 -o Main -I headers source/*.cpp
./Main
```

## Compile and Run (Windows)
Enter the following commands: 
```bash
g++ -std=c++20 -o Main -I headers source/*.cpp
Main
```
